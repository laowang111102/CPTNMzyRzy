# 毕业生就业管理系统 java300

## 项目概述

本项目是一款基于SSM框架与JSP技术的毕业生就业管理系统。系统旨在提高毕业生就业信息管理的效率和准确性，为学校、学生、教师及集团提供便捷的信息服务。

## 技术栈

### 后端

- Spring
- SpringMVC
- MyBatis（SSM）

### 前端

- JSP

### 数据库

- MySQL 5.7

### 开发工具

- IntelliJ IDEA

## 运行环境

- Servlet容器：Apache Tomcat
- 访问地址：[http://localhost:8080/jiuye/](http://localhost:8080/jiuye/)

## 功能模块

系统主要包括以下功能模块：

- 学生信息管理
- 教师信息管理
- 集团信息管理
- 就业信息发布与查询
- 系统管理

## 系统角色

- 超级管理员
- 学生
- 教师
- 集团

## 目录结构

```
project
├── src
│   ├── main

│   └── test
└── pom.xml
```

## 部署步骤

1. 环境准备：安装Java开发环境、配置MySQL数据库、安装Tomcat服务器。
2. 数据库配置：导入项目所需的SQL文件到MySQL数据库。
3. 项目构建：使用IntelliJ IDEA导入Maven项目，并执行`mvn clean install`命令。
4. 部署到Tomcat：将构建好的`war`包部署到Tomcat服务器的`webapps`目录下。
5. 启动Tomcat，访问系统。
